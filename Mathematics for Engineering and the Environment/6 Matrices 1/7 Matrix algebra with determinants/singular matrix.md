#### What is a
## Singular matrix
Basically if a matrix has a [[determinant]] equal to 0 it is singular, if not it is non-singular.

> ### $$ |A| = 0 $$ 
>> where:
>> $A=$ a singular matrix


> ### $$ |A| \neq 0 $$ 
>> where:
>> $A=$ a non-singular matrix
